# //项目文档

这是本游戏项目的开发文档，用于帮助开发者快速了解本项目中的一些约定性设定，已有成果以及未来工作计划等等。本文也可以在在开发中用作成员提供基本资料的查询使用。

## 项目基本设定

项目名称：（待定）

项目开发引擎：Godot

项目维度：2D

主要开发语言：C#

游戏类型：

* 局内策略战棋，局外角色扮演

美术风格：

* 黑白像素风，加上鲜红色的血液表现

关键特征/细节设定：

* 历史背景
* 反英雄主义
* 偏紧张压抑的游戏氛围
* 低血量设计，死亡即退场
* 瓦片格子
* 两伙人对打
* 得动脑子才能玩
* 失去攻击能力不等于阵亡，还可以互动
* 最终目的是活而不是赢
* 局内为局外服务
* 主角特权，没有战争迷雾
* 警戒设定
* 依靠卡攻击范围卡玩法
* 存在随机性的游戏方案
